# Assets

Drop the pipeline-editor screenshot here as **`editor.png`** (the one used in the
"Editor" section's scroll animation).

- Filename: `editor.png` (or update the `<img src>` in `index.html`).
- Recommended: ~2000 px wide, PNG or JPG, showing the full editor canvas.

Until the file exists, the framed section shows a placeholder ("Drop your editor
screenshot at assets/editor.png") instead of a broken image.
